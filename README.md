# Retail-Store-Sales-Analysis-sql
A real-world retail store database system built using SQL, implementing relational tables, foreign keys, and analytical queries for sales, orders, and customer insights.

# Project Overview

This project implements a Retail Store Database System using SQL.
It manages information about customers, products, categories, orders, and order items.
The database structure demonstrates the use of primary keys, foreign keys, relationships, and analytical queries for retail sales data.

# Database Tables

The project contains the following tables:

Customer – Stores customer information

Categories – Stores product category details

Product – Stores product details and stock information

Orders – Stores customer order information

OrderItems – Stores products included in each order
